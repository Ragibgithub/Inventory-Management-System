# Inventory-Management-System
The Inventory Management System is a desktop application designed to manage product stock, suppliers, and sales records efficiently. It helps businesses track inventory levels, update stock, generate reports, and maintain organized records in a simple and user-friendly interface.

---

## Features

- Add, update, delete, and view inventory items  
- Manage suppliers and product categories  
- Track stock quantities and availability  
- Purchase & sales record management  
- Database-based data storage  
- Simple and user-friendly interface  
- Secure and structured application workflow
